StayFinder Pro â€“ Full-Stack Hotel Booking Platform

GitHub Live Demo

Overview

StayFinder Pro is a modern, full-stack online hotel booking platform designed to help users discover, book, and manage unique stays. It features a beautiful, responsive user interface, a secure backend with robust user authentication, and host functionalities for property management.

Key Features

  • Secure Authentication: Full user sign-up and login functionality using JSON Web Tokens (JWT) for security and Bcrypt.js for password hashing.
  • Advanced Property Search: Explore properties with comprehensive filters for location, price, type, amenities, and ratings.
  • Interactive Map View: Visualize all property locations on an interactive map, powered by Leaflet.js.
  • Image Uploads: Hosts can upload multiple high-quality images for their properties, stored on Cloudinary.
  • Host Functionality: A dedicated dashboard for hosts to add, edit, and manage their property listings and view incoming bookings.
  • Real Booking System: A complete reservation flow allowing users to select dates and confirm bookings.
  • User Profile Management: Users can access a dedicated profile page to view their past bookings and manage their listed properties.

Technologies Used

Frontend

  • React & Vite
  • Tailwind CSS
  • Zustand (State Management)
  • React Query (Data Fetching)
  • React Router DOM
  • Leaflet.js (Maps)

Backend

  • Node.js & Express.js
  • Supabase
  • Cloudinary (Image Storage)
  • JSON Web Tokens (JWT)
  • Bcrypt.js (Password Hashing)
  • CORS & Dotenv

Deployment

The application is deployed with a decoupled architecture for performance and scalability:

  • The frontend React application is deployed on Vercel.
  • The backend Node.js API is deployed as a web service on Render.

CodeVerse - Real-Time Collaborative Code Editor

GitHub Live Demo

Overview

CodeVerse is a feature-rich, real-time collaborative code editor designed to streamline pair programming and technical interviews. It provides a synchronized coding environment where multiple users can write, edit, and review code together seamlessly.

Key Features

  • Real-Time Collaboration: Code and user presence are synchronized instantly across all connected clients using Socket.io.
  • Rich Code Editor: Powered by CodeMirror, offering syntax highlighting, auto-closing brackets, and line numbers for a superior coding experience.
  • Room-Based Sessions: Users can create or join unique rooms using generated UUIDs, ensuring private and secure collaboration sessions.
  • User Presence Avatars: Easily see who is currently in the room with creatively generated avatars for all participants.
  • Copy Room ID: A simple one-click button to copy the current Room ID for easy sharing.
  • Responsive Design: A clean and intuitive interface that works seamlessly across various devices and screen sizes.

Technologies Used

Frontend

  • React.js
  • CodeMirror (for the editor)
  • React Router DOM
  • Socket.io-client
  • React Avatar
  • React Hot Toast (for notifications)

Backend

  • Node.js
  • Express.js
  • Socket.io
  • Nodemon
  • UUID (for room IDs)
